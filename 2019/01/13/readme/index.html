博客搭建说明

2019-01-13

博客使用基于node.js的Hexo和zhwangart的theme，以及github.io和阿里云域名管理搭建完成。

The First Step

  • 拥有Github账号
  • 本地下载Hexo
  • Hexo配置关联Github(yourname.github.io)

The Second Step

  • Hexo找个主题
  • Hexo同步环境文件到Github分支
  • Hexo插件安装

The Last Step

  • 阿里云免费域名(不知道现在还有没有)
  • 配置域名解析到Github
